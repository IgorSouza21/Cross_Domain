to run the code in the terminal call the file main.py
passing as parameter the acronym of the classifier.

example: python main.py lr

the possible acronyms can be seen below:

nb -> Naive Bayes
dt -> Decision Tree
lr -> Logistic Regression
knn# -> k-Nearest Neighbors where # is the number of k, can be either 3, 5 or 7
wknn# -> Weighted k-Nearest Neighbors using the same number of k used in knn
svm -> Support Vector Machine with Linear Kernel
rf -> Random Forest
